# BaekjoonLog
## 간단 설명
1. 백준에서 본인의 풀이 코드를 넣어서 제출하고 채점 결과가 나온다.
2. 아래의 `json`을 `body`로 하여 미리 지정해둔 서버로 `POST` HTTP 요청을 보낸다.

```json
{
  "code": "1234\n",
  "submissionId": "88129297",
  "baekjoonId": "jwc5068",
  "problemNumber": "5073",
  "problemTitle": "삼각형과 세 변",
  "resultText": "컴파일 에러",
  "memory": "",
  "time": "",
  "language": "C++17 / 수정",
  "codeLength": "4",
  "submittedAt": "2025-01-03T15:50:50"
}
```
